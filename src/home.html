Turn your time into an endpoint.
Escalatex exposes /.well-known/escalatex so agents (and humans) can discover your escalation tiers, request priority help, and pay via Solana.
View provider profile View capabilities JSON
Self-hostable reference implementation. Payments buy priority review/response SLA, not guaranteed completion.
Quickstart
Create a request (idempotent):
curl -s -X POST ${location.origin}/.well-known/escalatex \
  -H 'Content-Type: application/json' \
  -H 'Idempotency-Key: <uuid>' \
  -d '{"subject":"Help me", "details":"…", "desired_tier":"24h"}' | jq
For development, use /demo for the interactive playground.
How it works
1) Client discovers capabilities → 2) Client posts request → 3) Server returns requires_payment → 4) Client pays with Solana Pay → 5) Provider gets notified and can acknowledge/complete.
